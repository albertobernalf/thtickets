El realidad si te esta tomando los anchos configurados, el problema es que por defecto el DataTable v1.10 es "responsive", es decir, todo table.dataTable tiene width: 100%.

Solución:

Podrías crear una clase para pisar esta configuración por defecto.

Ejemplo:

table.dataTable.dataTable_width_auto {
  width: auto;
}
